# DokodemoTV 使いかたガイド (How2Use)

「DokodemoTV」をご利用いただきありがとうございます。このドキュメントでは、アプリのインストール方法と基本的な使い方、およびチャンネルリストの編集方法について説明します。

## 1. アプリのインストール

ビルド済みのAPK（`app-debug.apk`）を端末にインストールしてください。

### 方法A: ADBを使用してインストール（開発者向け）
PCとAndroid端末を接続し、以下のコマンドを実行します。
```powershell
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### 方法B: 端末へ転送してインストール
1. `app-debug.apk` をGoogle Drive、USBメモリ、またはファイル転送アプリ等で端末に送ります。
2. 端末のファイルマネージャーでAPKを開き、「不明なアプリのインストール」を許可してインストールします。

## 2. 基本的な使いかた (現在のPoC版)

### 2-1. アプリの起動
アプリを起動すると、背景でストリーミング再生が自動的に開始されます。
現在のバージョン（Proof of Concept）では、**AbemaTVの特定チャンネル（例：One Pieceチャンネル）が固定で再生**される仕様になっています。

### 2-2. 再生コントロール
画面をタップすると、ExoPlayerの標準コントローラーが表示されます。
- **再生 / 一時停止**: センターのボタンで切り替えます。
- **全画面表示**: 右下の全画面アイコンで切り替え可能です。

## 3. チャンネルリストのカスタマイズ (今後の連携予定)

現在はコード内にURLが固定されていますが、今後のアップデートにより以下のファイルを編集することでチャンネルを自由に追加できるようになります。

1. **abema.txt**: AbemaTVのm3u8 URLリスト
2. **radiko.txt**: ラジオ（HLS形式）のURLリスト
3. **rakuten.txt**: 楽天サービス関連のURLリスト

### プレイリストの場所（将来の配置場所）
アプリの `app/src/main/assets/` フォルダ内にこれらのテキストファイルを配置することで、アプリが自動的に読み込むようになります。

---

## トラブルシューティング
- **映像が再生されない**: 端末がインターネットに接続されているか確認してください。
- **アプリがクラッシュする**: メモリ（RAM）が不足している可能性があります。他のアプリを閉じてから再起動してください。
