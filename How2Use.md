# DokodemoTV 使いかたガイド (How2Use)

「DokodemoTV」をご利用いただきありがとうございます。このドキュメントでは、アプリのインストール方法と基本的な使い方、およびチャンネルリストの編集方法について説明します。

## 1. アプリのインストール

ビルド済みのAPK（`app-debug.apk`）を端末にインストールしてください。

### 方法A: ADBを使用してインストール（開発者向け）
PCとAndroid端末を接続し、以下のコマンドを実行します。
```powershell
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

### 方法B: 端末へ転送してインストール
1. `app-debug.apk` をGoogle Drive、USBメモリ、またはファイル転送アプリ等で端末に送ります。
2. 端末のファイルマネージャーでAPKを開き、「不明なアプリのインストール」を許可してインストールします。

## 2. 基本的な使いかた

### 2-1. アプリの起動
アプリを起動すると、背景でストリーミング再生が自動的に開始されます。
デフォルトでは、**AbemaTVの特定チャンネル（例：One Pieceチャンネル）**が再生されます。

### 2-2. 再生コントロールとメニュー
画面をタップすると、標準コントローラー（再生/一時停止、全画面など）が表示されます。
また、リモコン（D-Pad）や画面操作で以下のUIを利用できます。

- **オーバーレイメニューの表示/非表示**: 画面上部のトグルボタン、またはリモコンの **下に移動（D-Pad Down）** を押すことで、トップメニュー（「Load Folder」「Hide Menu」などのボタン）の表示/非表示を切り替えられます。
- **チャンネルリストを開く**: 画面左上のメニュー（≡）ボタンをタップするか、リモコンの **上に移動（D-Pad Up）** を押すことで、画面下部からチャンネルリスト（BottomSheet）を開くことができます。

## 3. チャンネルリストのカスタマイズ (プレイリストの読み込み)

現在のバージョンでは、端末内のフォルダからプレイリストファイルを読み込み、チャンネルを自由に追加・切り替えできるようになっています。

### 3-1. プレイリストファイルの用意
端末本体のストレージやSDカード等にフォルダを作成し、その中に `.txt` または `.csv` 形式のプレイリストファイルを配置します。
読み込み時、**ファイル名がそのままカテゴリ名（タブ名）** として扱われます（例: `abema.txt` とすると「abema」タブになります）。

ファイルの中身は以下のフォーマットで、1行に1チャンネルずつ記述します。

**カンマ区切りのフォーマット:**
```text
チャンネル名,アイコン画像のURL,ストリームURL
```
*(例)*
```text
Abema News,https://example.com/icon.png,https://.../playlist.m3u8
My Channel,,https://.../stream.m3u8
```
※ アイコン画像URLは省略可能です（その場合は `チャンネル名,ストリームURL` または `チャンネル名,,ストリームURL` とします）。ストリームURLのみ記述することも可能です。

### 3-2. フォルダの読み込み手順
1. アプリ右上の **「Load Folder」** ボタンを押します（ボタンが見えない場合はメニューを表示してください）。
2. Android標準のファイル選択画面が開きます。
3. プレイリストファイルを入れたフォルダを探して選択し、**「このフォルダを使用」** をタップしてアクセス権限を許可します。
4. 読み込みが完了すると、チャンネルリストパネル（D-Pad Upで表示）にファイルごとのタブが作成され、選択したチャンネルの再生が可能になります。

---

## トラブルシューティング
- **プレイリストが読み込まれない**: 選択対象が「ファイル」ではなく「フォルダ」自体であることに注意してください。また、拡張子が `.txt` または `.csv` であることを確認してください。
- **映像が再生されない**: 端末がインターネットに接続されているか確認してください。ストリームURLの有効期限が切れている場合もあります。
- **アプリがクラッシュする**: メモリ（RAM）が不足している可能性があります。他のアプリを閉じてから再起動してください。
