# DokodemoTV Android版 開発マイルストーン (Todo)

## Phase 1: PoCの動作確認と基盤構築
- [ ] Android Studio等のIDE環境でのプロジェクトビルドと実機（Android端末/TVエミュレータ）での起動確認
- [ ] 外部テキスト（例: `abema.txt` 等のダミーデータ）をローカルアセットから読み込み、リスト化するロジックの実装
- [ ] ExoPlayer(Media3)を用いたHLS(m3u8)ストリームの基本再生処理の安定化

## Phase 2: UI/UXの構築（TV/モバイル両対応）
- [ ] モバイル向けUI：スワイプ操作や各種タップジェスチャーによる直感的なチャンネル切替インターフェースの実装
- [ ] Android TV/プロジェクター向けUI：リモコンのD-Pad（十字キー）によるフォーカス移動とシームレスなチャンネル・モード切替への対応
- [ ] テレビ欄（電子番組表: EPG）を表示する簡易ダイアログ／グリッド画面の実装

## Phase 3: 応用・拡張機能の追加
- [ ] 録画・ダウンロード機能（ExoPlayer DownloadManager等を活用したストリームの保存）の試験実装
- [ ] 視聴履歴およびお気に入り等の各種ユーザーデータを永続化（Roomデータベース）する仕組みの導入
- [ ] 番組表(EPG)データを公式API等から取得し、番組情報とm3u8ストリームURLを紐づけてUI上から直接チャンネル選択・再生させる本格連携機能の構築
- [ ] ExoPlayerのインターセプタやエフェクト連携（映像・音声）を利用した簡易イコライザー・プリセット機構の導入

## Phase 4: リリース・本番に向けた準備
