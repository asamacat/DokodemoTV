

# DokodemoTV

**AndroidおよびAndroid TV向けの拡張性に富んだHLSストリーミングプレーヤー**



-----

## 概要
**DokodemoTV**は、専用テキストリストベースのHLS（`.m3u8`）プレイリストを利用したインターネットストリーミングTVプレーヤーです。スマートフォンやタブレットのみならず、プロジェクターやセットトップボックスなどの**Android TV端末でも快適に動作**するようフルネイティブで構築されています。

###
プロジェクター垂れ流しや使ってないタブレットに垂れ流しのために作りました。


-----

## 現在の機能 (PoC)

  * **HLSストリーミング再生:** Android標準のMedia3 (ExoPlayer) エンジンを採用し、安定したHLS (`.m3u8`) プレイリスト再生を実現。
  * **ローカルプレイリスト読み込み:** `assets` フォルダ内のテキストファイル（`abema.txt` 等）から動的にチャンネルリストをパース。
  * **マルチデバイス対応基盤:** スマートフォン・タブレットに加え、Android TVのLeanbackランチャーに対応したマニフェスト構成。
  * **シンプルなチャンネル切替:** リストからの選択のみで即座に再生が開始される直感的なUI。
