# DokodemoTV for Android & Android TV

## 概要
「DokodemoTV」は、専用テキストリストベースのHLS（.m3u8）プレイリストを利用した、シンプルかつ拡張性に富んだインターネットストリーミングTVプレーヤーです。
PC（Windows 11）版で培われた豊富な機能を継承し、モバイルデバイスだけでなく、プロジェクターやセットトップボックスなどの「Android TV端末」でも快適に利用できるようフルネイティブ構築されています。

## 現在の機能 (PoC)
- **HLSストリーミング再生**: Android標準のMedia3 (ExoPlayer) エンジンを採用し、安定したHLS (.m3u8) プレイリスト再生を実現。
- **ローカルプレイリスト読み込み**: `assets` フォルダ内のテキストファイル（`abema.txt` 等）から動的にチャンネルリストをパース。
- **マルチデバイス対応基盤**: スマートフォン、タブレットだけでなく、Android TVのLeanbackランチャーにも対応したマニフェスト構成。
- **シンプルなチャンネル切替**: リストから選択するだけで即座に再生が開始される直感的なUI。

## 今後のロードマップ
- **Android TV / プロジェクター最適化**: D-Pad（十字キー）操作への完全チューニングと、大画面向けの専用UI構築。
- **録画・ダウンロード機能**: ライブストリームのローカル保存機能（TS/MP4形式）の実装。
- **番組表（EPG）連携**: 外部API等から取得した番組情報とストリームURLの完全統合。
- **ユーザーデータ永続化**: 視聴履歴やお気に入り機能の追加（Room Database利用）。
- **画質・音質調整**: イコライザーやバッファリング設定などのカスタムオプションの導入。

## 技術スタック
- **開発言語**: Kotlin
- **UIアーキテクチャ**: Jetpack Compose (スマートフォン/タブレット向け) & Compose for TV (大画面・Android TV向け)
- **メディア再生モジュール**: `androidx.media3:media3-exoplayer` 周辺システム
- **データ管理**: SQLite / Room Database (履歴・ユーザー設定等)
