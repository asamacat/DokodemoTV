# DokodemoTV for Android & Android TV

## 概要
「DokodemoTV」は、専用テキストリストベースのHLS（.m3u8）プレイリストを利用した、シンプルかつ拡張性に富んだインターネットストリーミングTVプレーヤーです。
PC（Windows 11）版で培われた豊富な機能を継承し、モバイルデバイスだけでなく、プロジェクターやセットトップボックスなどの「Android TV端末」でも快適に利用できるようフルネイティブ構築されています。

## 主な機能 (ゴール想定)
- **TV/モバイル マルチプラットフォームUI**: モバイル画面のタッチ操作への対応にとどまらず、Android TVの赤外線リモコンやBluetoothプロジェクターリモコン（D-Pad操作）にも完全チューニングされたUIを提供。
- **軽量かつ堅牢なストリーミング**: Android標準のMedia3 (ExoPlayer) エンジン採用による、高速・安定したHLSプレイリスト再生。
- **ローカルプレイリスト管理機構**: アプリ内の `assets` フォルダ、または外部ストレージから動的にチャンネルリスト（txt / m3u等）をパースし展開する機能。
- **テレビ欄（番組表）表示**: 直感的に視聴したいチャンネルや番組を探せる、リモコン対応の番組表グリッドインターフェース。
- **録画・ダウンロード対応**: 視聴中のライブストリームをローカルストレージにTSやMP4形式で保存し、電波のないオフライン環境でも再生に備える機能。

## 技術スタック
- **開発言語**: Kotlin
- **UIアーキテクチャ**: Jetpack Compose (スマートフォン/タブレット向け) & Compose for TV (大画面・Android TV向け)
- **メディア再生モジュール**: `androidx.media3:media3-exoplayer` 周辺システム
- **データ管理**: SQLite / Room Database (履歴・ユーザー設定等)
